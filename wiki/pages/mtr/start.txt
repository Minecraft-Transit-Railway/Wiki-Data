====== Minecraft Transit Railway ======

**Minecraft Transit Railway** is a [[https://minecraft.wiki/Mods| Minecraft mod]] based on Hong Kong's MTR, the London Underground, and the New York Subway. It adds trains into the game along with other miscellaneous blocks and items. With this mod, it is possible to build a fully functional railway system in your world!

This mod is available for Minecraft Forge and Fabric versions 1.16, 1.17, 1.18, and 1.19 (1.20 is also available in Alpha stages, please create backups before using this version).

Please report any issues or bugs that you find; that would be greatly appreciated! Refer to the [[https://github.com/jonafanho/Minecraft-Transit-Railway/issues| GitHub issues tab]] or the [[https://github.com/jonafanho/Minecraft-Transit-Railway/wiki/Todo-List| todo list]] to see currently known issues.
