====== 編輯指南 ======

===== 翻譯頁面 =====

導航到要翻譯的頁面，從左側選擇要翻譯成的語言，然後單擊右側的“創建該頁面”按鈕。

應該自動拷貝要翻譯的頁面的代碼。您可以單擊“Toggle syntax / visual editor”按鈕來選擇您偏好的編輯方法（代碼編輯或可視化編輯）。

如果左邊未包含您想要翻譯成的語言，請與管理員聯繫。

===== 創建新頁面 =====

首先為您的頁面想一個全英文的標題，如 "Edit Guideline"。接下來，選擇一個頁面適合的類別。

| mtr | MTR 本體遊玩方面的指引 |
| mtr->dev | 為 MTR 開發資源包或擴展 |
| mtr_letsplay | 官方 Let's Play 服務器 |
| mtr_extra | 社群製作的資源包 |
| mtr_addon->lu | 倫敦地鐵擴展 |
| mtr_addon->nte | NTE 藍星社交通擴展 |
| mtr_addon->msd | MSD 車站裝飾 |
| playground | 用來自己試試編輯功能 |

接下來，您就可以創建頁面了。按鈕在本頁最底下。

創建頁面後，您可以單擊“Toggle syntax / visual editor”按鈕來選擇您偏好的編輯方法。將頁面標題寫在第一行，並使用“H#”按鈕（如果您用可視化編輯器）將其設置為“Heading 1”，或將其設置成“======頁面標題======”這樣（如果您用代碼編輯器；請注意這裡必須是 12 個等號）。您可以在此標題中使用任何語言，而不限於頁面名稱中那樣只能使用英語。

添加標題後，您可以在下方編寫任何所需內容。

您應該用英語寫這個新頁面的內容。如果您想先用別的語言寫，請您先在這個英語的頁面上寫一段說明這個情況的話，保存，然後照著上述“翻譯頁面”的操作來做。

如果您碰巧喜歡使用Markdown，可以將Markdown內容放在%%<markdown></markdown>%%之間。請注意Markdown與可視化編輯器不兼容。
